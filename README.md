# FUTURE_DS_01
# 📊 Business Sales Performance Analytics

## Project Overview

This project analyzes business sales data using Power BI to uncover key revenue trends, customer purchasing patterns, product performance, and geographical sales distribution. The dashboard provides interactive visualizations that help businesses make data-driven decisions and improve overall sales performance.

---

## 🎯 Objectives

* Monitor overall sales performance.
* Analyze revenue trends across different time periods.
* Identify top-selling products.
* Understand customer purchasing behavior.
* Evaluate country-wise sales distribution.
* Generate actionable business insights.

---

## 🛠️ Tools & Technologies

* Power BI Desktop
* Power Query
* DAX (Data Analysis Expressions)
* Microsoft Excel / CSV Dataset

---

## 📈 Dashboard Features

### Key Performance Indicators (KPIs)

* Total Revenue
* Total Quantity Sold
* Total Customers
* Total Invoice Count

### Visualizations

* Monthly Revenue Trend Analysis
* Revenue by Product Description
* Revenue by Country (Map Visualization)
* Customer Activity Analysis
* Revenue, Quantity, and Unit Price Comparison
* Interactive Filters and Drill-Down Analysis

---

## 🔍 Key Insights

* Revenue showed noticeable fluctuations throughout the year.
* A small number of products contributed significantly to total revenue.
* Customer purchasing activity varied across different months.
* Certain countries generated higher sales volumes than others.
* Product-level analysis helped identify high-performing inventory items.
